# Wortkklar

Минималистичный русско-немецкий и немецко-русский переводчик для изучения языка.

## Идея

Wortkklar не только переводит, но и объясняет структуру языка:

- показывает артикль и род существительных;
- показывает множественное число;
- выводит три основные формы глагола;
- отмечает вспомогательный глагол и управление;
- объясняет различия употребления;
- даёт живые примеры с переводом;
- позволяет сохранять слова для повторения.

## Текущий статус

Первый UI-прототип. Перевод и грамматический анализ пока представлены демонстрационными данными.

## Запуск

1. Установить Node.js 20 или новее.
2. Выполнить `npm install`.
3. Выполнить `npm run dev`.
4. Открыть `http://localhost:3000`.
